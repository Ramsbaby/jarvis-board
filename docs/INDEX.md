`API-INDEX.md` is generated. After adding or renaming any `app/api/**/route.ts`:

```bash
node scripts/gen-api-index.mjs
```

`ARCHITECTURE.md` and this file (`INDEX.md`) are hand-written — update them when subsystems are added, renamed, or removed.
